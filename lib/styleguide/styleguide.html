RubyAU Style Guide

The RubyAU Style guide is designed as a reference to review and implement styles on the Ruby AU homepage. The CSS styles are built using the tachyons library, and code can be copied and implemented into any HTML files.

Typefaces

There are two typefaces used across the Ruby AU webpage; Poppins & Anonymous Pro

Poppins - primary

Poppins, is a geometric sans serif typeface. Each letterform is nearly monolinear, with optical corrections applied to stroke joints where necessary to maintain an even typographic colour.

Anonymous Pro - secondary

Secondary Typeface Anonymous Pro, is a family of four fixed-width fonts designed especially with coding in mind. It is inspired by Anonymous 9, a freeware Macintosh bitmap font developed in the mid-'90s that was intended as a more legible alternative to Monaco.

Typographic Measure - Line length

An effective line length helps greatly in making text readable.

To help enforce a consistent measure on all body copy, we use the convention, that section tags provide this constraint.

Use a section tag around your body copy and any elements that you want to fit within the line measure. This approach is neutral to if you place a single section inside your body or define multiple sections.

Just avoid nesting sections, to avoid the restriction applying twice, use a div or article tag instead.

Elements that you want to run edge-to-edge of the screen, like background images should be placed outside of section tags. Most elements, however, will likely be within this tag.

Our measure is achieved with a max-width of 660px.
